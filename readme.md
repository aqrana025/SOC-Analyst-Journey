### **The "Why" Behind This Project**
I’ve always believed that in cybersecurity, knowing *how* a tool works is only half the battle—the real skill is knowing **why** an alert triggered and what to do next. I created this repository to document my transition from a student of security to a hands-on practitioner.

> This isn't just a list of labs; it's my personal "firing range" where I simulate real-world attacks, break things, and then learn how to detect and fix them.

---

### **My Focus**
* **SIEM & Monitoring:** Diving deep into **Splunk (SPL)** and **Wazuh** to separate the signal from the noise.
* **Log Analysis:** Learning the language of **Windows Event Logs (Sysmon)** and **Linux Auth logs**.
* **The Analytical Mindset:** Mapping every lab to the **MITRE ATT&CK** framework and **NIST** incident response steps.
